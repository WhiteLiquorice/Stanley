Reviewer Installation & Validation Guide

  1. 1.
    Extract the Extension ZIP Extract the contents of stanley-extension.zip to a local directory.
  2. 2.
    Load the Extension in Chrome Navigate to chrome://extensions in Google Chrome, toggle **Developer mode** in the top-right, click **Load unpacked**, and select the extracted extension folder.
  3. 3.
    Run the Windows Host Daemon Run the downloaded stanley-setup.exe. This starts the local companion Native Messaging listener.
  4. 4.
    Sign In and Authenticate Click the extension icon in the toolbar, select **Sign In**, and authenticate using the reviewer credentials listed above.
