## Full Description

**Stanley Browser Butler** is an AI-powered browser automation tool for business professionals who need to complete repetitive, multi-step web tasks without writing code.

**How It Works**

Tell Stanley what you need done in plain English. Stanley's AI interprets your instruction, builds a step-by-step automation plan, and shows it to you for review before executing anything. You stay in control — Stanley acts only when you confirm.

**What You Can Do**

- Automate any sequence of browser actions: navigate, click, fill forms, submit, wait
- Extract and collect structured information from web pages
- Manage multiple browser tabs within a single workflow
- Review the automation plan before it runs — no surprises
- Get a desktop notification the moment a workflow finishes so you can step away

**Built for Business Productivity**

Stanley is designed for professionals who spend time on the same browser workflows every day — logging into portals, pulling reports, updating records, checking dashboards. If you can describe it in a sentence, Stanley can automate it.

**How It's Set Up**

Stanley is a companion extension — it works alongside a lightweight local application (the Stanley daemon) installed on your Windows or Mac machine. The extension connects to this local application via Chrome's Native Messaging API, a secure local-only communication channel. All automation runs on your machine. AI-powered workflow planning uses an authenticated cloud service for natural language interpretation only.

**Privacy First**

Your credentials, workflow history, and session data never leave your device. The only data sent to external services is the text of your workflow prompt and, when needed for element resolution, a compressed screenshot of the active page. No browsing history is collected or transmitted. See the privacy policy for full details.
